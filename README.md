# Portafolio de Machine Learning: Fundamentos y Práctica Aplicada

Bienvenido a mi repositorio. Este espacio fue creado con un objetivo principal: **traducir la teoría matemática y estadística en código funcional mediante la repetición y la práctica empírica.**

Aquí documento mi proceso de aprendizaje continuo, desarrollando modelos predictivos para asimilar a profundidad el ciclo de vida de los datos.

## Enfoque del Repositorio
Más que especializarme en una sola industria por ahora, estos proyectos exploran la versatilidad de los algoritmos clásicos aplicados a distintos conjuntos de datos. Mi objetivo es afianzar la lógica detrás de cada herramienta.

Mi metodología de trabajo documentada en cada libreta (`.ipynb`) se centra en:
* **Preprocesamiento y Transformación:** Limpieza de datos, codificación de variables categóricas y división estratégica de muestras (Train/Test split).
* **Implementación de Modelos:** Construcción de algoritmos de regresión y clasificación (Ej. Regresión Lineal Múltiple, Regresión Logística, Regresión Lineal).
* **Interpretación:** El esfuerzo por traducir los coeficientes y las matemáticas en conclusiones lógicas y visuales.

## Stack Tecnológico
* **Lenguaje Central:** Python
* **Procesamiento de Datos:** Pandas, NumPy
* **Modelado Estadístico / ML:** Scikit-Learn
* **Visualización:** Matplotlib, Seaborn
